The Legitimacy Discipline

The Shuffle Test

Ask, tile by tile: could this swap places with a differently-sized tile and lose nothing? If yes, the sizing is decorative, not bento — go back and either find the real signal or flatten the tiles to a plain list.

Real Hierarchy Signals, Named

Three legitimate reasons a tile earns extra space: importance/significance, recency, and data density (a chart that genuinely needs the height or width to render honestly). "Visual variety" and "it looked empty" are not on this list.

When Not to Use This Archetype

If every item is genuinely comparable in importance, recency, and density, don't invent size differences to look organized — use a plain list, magazine-well's well band, or two-column-academic instead.
