Layout-Factory · View: Concept slide-master-deck-grid

The concept: this archetype operates one level up from any single slide. It fixes the title-zone and footer/citation-zone margins so they land in the exact same place, at the exact same size, on every slide in a deck — while leaving the body zone free to be whatever that slide's own content archetype needs. A slide built on slide-rule-of-thirds and a slide built on poster-broadside can sit back-to-back in the same deck and still feel like one continuous document, because the frame around them never moves. Best suited to any multi-slide deck that needs to feel like one authored artifact rather than a stack of loose pages: pitch decks, board decks, conference talks, investor updates.

When Not to Use This Archetype

A one-off single slide

If there's no deck around it — no set of slides that need to feel like one continuous document — there's no meta-grid to be consistent about. Reach for slide-rule-of-thirds or poster-broadside directly; this archetype only earns its keep across multiple slides.

Structural rule

The masthead and footer zone heights are fixed the moment the deck starts. No slide is allowed to shrink or grow them to make room for its own content — if a slide's exhibit needs more space, it gets a second slide, not a taller body zone at the fixed zones' expense.

Editorial devices this archetype favors

A numbered footer citation on every slide (page number + a running confidentiality or source line) and nothing else — no drop caps, no sidebars, no pull quotes as deck-level features. Those belong to whichever per-slide archetype fills the body zone, not to the fixed frame around it.
