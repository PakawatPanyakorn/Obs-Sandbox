The Concept

One Path, One Action

This archetype exists to move attention, not to be read start-to-finish. The eye tracks an actual Z: logo (1) → nav CTA (2) → a diagonal sweep through the supporting visual and proof copy (3) → the closing call-to-action (4), which is deliberately the single most visually treated element on the page.

Unlike an editorial layout, nothing here is competing to be "read most" — everything upstream of the closing CTA exists only to make that one click feel like the obvious next move. Size, color, and shadow are spent almost entirely on the button at point 4, not on a hero photo.

"We didn't redesign the product page. We just made sure everything pointed at the same button."— a made-up but representative line of the register this pattern is built for

Use this when you have exactly one conversion action and the whole page's job is steering toward it: a product landing page, a waitlist or beta signup, a single-offer marketing page, a campaign landing page tied to one ad.

Stand-in for a product shot or hero illustration. It supports the diagonal sweep but is NOT the page's dominant visual — that treatment is reserved for the closing CTA below.

When Not to Use This Archetype

Content meant to be read start-to-finish

An essay, a report, a documentation page — anything where comprehension of the whole matters more than one click. Reach for single-column-editorial or asymmetric-split instead, where the structure serves reading, not conversion.

Structural rule that keeps this from becoming a dashboard

Exactly one region gets --lf-shadow-hero/--lf-gradient-hero: the closing CTA. If a second element (a hero photo, a stat tile) starts competing for that same visual weight, the "one action" premise collapses.

Editorial devices this archetype favors

One short social-proof testimonial in the lead, styled as a pull-quote. Microcopy under the CTA (e.g. "no credit card required") to lower last-second friction. No footnotes, no sidebars — those belong to reference material, not a conversion page.
