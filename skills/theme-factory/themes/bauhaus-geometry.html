Unread Count Circle = closed, finite quantity. Humans read a red dot as "something needs attention" before seeing the number. Use only for counts — never labels.

Subscription Tier Hexagon = structured hierarchy. One hex implies a grid of tiers above and below. Use for plan levels, skill ranks, or any badge that belongs to a system of levels.
Launch
Call to Action Arrow = momentum, move forward. The pointed tip makes the badge itself a visual verb. Use for primary CTAs, "what happens next", or feature launches where action is the point.
Review
Pipeline Stage Chevron = middle of a sequence. Open on both sides — it receives from the left and passes to the right. Use for pipeline stages, breadcrumb steps, or any status that lives between two others.
New
Scarcity / Rarity Tag Diamond = rare, valuable, exclusive. The angular form costs visual attention — use sparingly. Best for limited releases, premium features, or "only X left" signals.
Admin
Permission / Role Badge Shield = granted authority. Someone gave you this — it implies a trust hierarchy. Use for user roles, access levels, moderation status, or verified/certified labels where authority was earned.
